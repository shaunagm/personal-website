Kybern

Kybern is/was a project exploring how we can use technology to enable customizable, collective self-governance online.

I paused work on this project after discovering Amy Zhang's PolicyKit, a better architected and more flexible version of the same concept. I highly recommend checking it out. I hope to return to Kybern, at least in spirit, and continue exploring how technology can enable governance.

The site itself is currently down thanks to Heroku removing their free tier, but you can see the source on Github. You can also see a demo via this blog post or watch any of these three presentations I gave which mention/feature Kybern:

  • A Python Library for Digital Self-Governance for the Metagov Seminar
  • Code as Contestable Law, IASC Knowledge Commons (video, panel)
  • Engineering Responsive Governance, IASC Knowledge Commons (video, panel)
